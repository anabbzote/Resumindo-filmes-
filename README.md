# 🎬 Resumindo Filmes

**Resumindo Filmes** é uma plataforma moderna que oferece resumos interativos e rápidos de filmes disponíveis em grandes plataformas de streaming como Netflix, Prime Video, Disney+ e outras.

![Logo Resumindo Filmes](logo.png)

## 🚀 Sobre o Projeto

A proposta é fornecer trailers curtos ou cenas principais dos filmes em um ambiente visualmente envolvente e organizado. É voltado para usuários que querem decidir o que assistir sem perder tempo.

## 🧠 Funcionalidades

- ✅ Exibição de trailers e resumos de filmes em carrossel horizontal (estilo catálogo);
- ✅ Layout responsivo e moderno;
- ✅ Conteúdo patrocinado por grandes plataformas de streaming;
- ✅ Monetização via anúncios leves e não-invasivos.

## 🖼️ Interface

A interface principal inclui:
- Logotipo da plataforma;
- Apresentação com textos explicativos;
- Lista de vantagens;
- Seção com carrossel de vídeos (YouTube embutido).

## 📁 Como usar

1. Faça o download ou clone este repositório.
2. Coloque os arquivos em uma pasta:
   - `index.html`
   - `logo.png`
3. Abra o arquivo `index.html` em seu navegador ou hospede online com:

### 🌐 Hospedagem sugerida

- [Netlify](https://netlify.com) (arraste e solte)
- [Vercel](https://vercel.com)
- [GitHub Pages](https://pages.github.com)

## 🧩 Exemplo ao vivo
